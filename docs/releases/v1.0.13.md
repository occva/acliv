# v1.0.13

## Changes

更新内容:

- 新增 Provider Paths 管理入口，可查看各 provider 的数据路径，并支持手动保存、重置路径。
- 桌面端支持选择 provider 文件夹，Windows 可在文件管理器打开，macOS 可在访达打开。
- macOS 桌面端支持用 Terminal 打开会话恢复命令，并优化工作目录识别。
- 修复切换到无数据 provider 时仍显示上一个 provider 缓存会话的问题。
- 优化 Provider Paths 和侧栏统计区域的视觉密度，弱化分隔线并统一整体观感。
- 补充内置更新按钮策略文档，明确桌面端和 Web 端后续更新方案。
- 完善发布流程校验，确保发布说明必须人工维护，避免脚本自动生成 Changes。

Updates:

- Added Provider Paths management for viewing, saving, and resetting each provider data folder.
- Added desktop folder selection and open-folder support, including Finder integration on macOS.
- Added macOS Terminal support for session resume commands with better working-directory handling.
- Fixed stale session lists when switching to a provider with no local data.
- Refined Provider Paths and sidebar stats styling with softer dividers and a more consistent look.
- Added the in-app update button strategy document for desktop and web update behavior.
- Hardened release validation so release Changes stay manually written instead of script-generated.
