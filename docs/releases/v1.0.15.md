# v1.0.15

## Changes

更新内容:

- 新增 Pi 会话来源，支持读取 `~/.pi/agent/sessions` 下的会话记录。
- 支持展示 Pi 的工具调用、工具结果、思考内容和会话树状父子关系。
- 优化索引自动刷新逻辑，减少无变化时的重复扫描和页面重算。
- 改进索引变更检测，新增、更新和删除会话文件都能触发正确刷新。
- Web 和 Docker 部署增加 Pi 数据目录映射，便于容器环境读取 Pi 会话。
- 版本更新到 `1.0.15`。

Updates:

- Added Pi as a session provider, reading sessions from `~/.pi/agent/sessions`.
- Added support for Pi tool calls, tool results, thinking content, and parent-child message links.
- Optimized automatic index refresh to avoid repeated scanning and UI recomputation when nothing changed.
- Improved index change detection so added, updated, and deleted session files trigger the correct refresh.
- Added Pi data directory mapping for Web and Docker deployments.
- Updated the app version to `1.0.15`.
