# v1.0.12

## Changes

更新内容:

- 优化首屏消息加载，减少启动时对慢接口的等待，消息列表打开更快。
- 修复搜索索引读取失败时的错误处理，避免 500 和重试循环拖慢会话加载。
- 修复 macOS 桌面端字体大小不一致的问题。
- 修复 Windows 发布包构建流程，恢复 exe、setup.exe 和 msi 三类 Windows 资产发布。
- 完善发布自动化的版本同步与校验，降低桌面端、Web 端和 Cargo 元数据版本不一致的风险。

Updates:

- Improved initial message loading by removing waits on slower startup requests and reducing retry work.
- Fixed indexed search message read failures so missing index data no longer causes 500 retry loops.
- Fixed inconsistent font sizing on macOS desktop.
- Fixed the Windows release packaging flow and restored exe, setup.exe, and msi assets.
- Hardened release automation version syncing and validation across desktop, web, and Cargo metadata.
